Physical Model

The model features six women who has contibuted to make a change in the society. It displays basic yet valuable information about these change makers along with moral lessons from their lives which benifits everyone who reads them. The model is made with a creative touch which also conatins the ORcodes that act as a gateway to this E-version.

E-version

The QR codes in the Physical Model act as gateways to this Website. This website has been sololey built by me(Sushaanth.K) with an effort of integrating technology into this meaningful project . The E-version provides more information about the matter that cannot be put into place in the Physical Model.

The Message

This effort taking project in overall carries a deep message to all audience. This project is made to with an aim to highlight the social issues that women has been facing for centuries.
